Introduction
================
ESP32-C6-LCD-1.47 is a high-performance microcontroller development board supporting 2.4GHz Wi-Fi 6 and Bluetooth BLE 5. It integrates 4MB SiP Flash and a 1.47-inch LCD color display, ensuring smooth operation of LVGL and other graphical interfaces. With abundant peripheral interfaces, it enables rapid development of ESP32-C6-based HMI and similar applications.

Features
===========

.. _key-features:

1. Equipped with high-performance and low-power RISC-V 32-bit dual-core processors: up to 160MHz for the high-performance core and up to 20MHz for the low-power core
2. Supports 2.4GHz Wi-Fi 6 (802.11ax/b/g/n) and Bluetooth 5 LE, with onboard high-efficiency antenna
3. Integrates 320KB ROM, 512KB HP SRAM, 16KB LP SRAM, and 4MB flash storage
4. Equipped with 1.47-inch LCD screen, 172×320 resolution, 262K color display
5. Rich GPIO interfaces, built-in full-speed USB to serial function
6. Onboard Micro SD card slot, expandable storage for pictures and files
7. Flexible clock management and various low-power modes for multiple application scenarios
8. Built-in RGB LED, presenting dazzling lighting effects together with a transparent acrylic interlayer
